def calculate_ndwi(image):

    return image.addBands(ndwi)

We are going to design a system that processes YouTube videos about educational content (specifically biology in this case) and generates detailed student and parent reports, career path suggestions, and related course recommendations. The system will use AI (Gemini) for content analysis and integrate with various data sources for career mapping and course suggestions.
### System Architecture Overview
1. **Input**: YouTube video URL
2. **Transcript Extraction**: Using YouTube API or other methods to get the video transcript.
3. **AI Analysis**: Using Gemini to analyze the transcript and extract key topics, relate them to real-world applications, and suggest career paths.
4. **Career Mapping**: Match the topics with career options from a database (like İŞKUR, BTK, etc.).
5. **Course Recommendation**: Fetch related courses from BTK Akademi or similar platforms.
6. **Report Generation**: Create two reports: one for the student (career-oriented) and one for the parent (guidance-oriented).
7. **Output**: Present the reports in a user-friendly format (PDF, HTML, etc.).
### Step-by-Step Implementation
#### 1. Transcript Extraction
- Use the YouTube Data API to get video details and captions.
- If captions are not available, use speech-to-text (like Google Cloud Speech-to-Text) to generate a transcript.
#### 2. AI Analysis with Gemini
- We will use the Gemini API to analyze the transcript. The prompt should be designed to extract:
  - Main topics
  - Real-world applications
  - Suggested career paths
  - Required skills
  - Relevance to the job market
Example prompt for Gemini:
```
Analyze the following video transcript about Protista. Extract the main topics, real-world applications, and suggest career paths that relate to these topics. Also, identify the skills needed for these careers and how the topics in the video are applied in real-world scenarios.
Transcript: {transcript}
```
#### 3. Career Mapping
- We have a database (or predefined list) of careers with associated topics and skills.
- The AI analysis will output a list of topics and skills. We then match these to careers in our database.
- We can use İŞKUR's occupation database and BTK's skill matrix for accuracy.
#### 4. Course Recommendation
- Query BTK Akademi's API (if available) or a predefined list of courses to find matches for the identified topics and skills.
#### 5. Report Generation
- **Student Report**:
  - Introduction: How the video content relates to real-world careers.
  - Main topics from the video.
  - Career paths (with details: job role, how the topic is applied, companies, etc.).
  - Steps to build a career (short-term and long-term).
  - Recommended courses (from BTK Akademi).
- **Parent Report**:
  - Introduction: Importance of the topic for the child's future.
  - Long-term plan: How to support the child from high school to career.
  - Practical activities to do at home.
  - Guidance on technology and resources.
